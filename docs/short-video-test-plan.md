# ショート動画検証プラン

## 目的

- ざぁこカロリーの「普通の食事記録アプリなのに、コメント欄だけキャロリンが煽ってくる」体験に反応があるか確認する。
- 結果画面のスクショ映え、キャロリンの見え方、短いAIコメントの刺さり方を検証する。
- 事前登録CTAに進みたいと思う人がいるかを見る。

## 反応を見る指標

- 3秒視聴維持率
- 最後まで見られた割合
- コメント欄で「使いたい」「面白い」「キャロリン好き」などが出るか
- 保存数、シェア数
- プロフィール遷移やLP遷移
- 仮CTAへの反応

## 撮影メモ

- 画面録画はスマホ縦長で行う。
- 1本あたり10〜20秒に収める。
- 解析中画面は長く見せすぎず、0.5〜1.0秒程度にする。
- 結果画面ではAIコメントが読める時間を最低2秒取る。
- CTAは最後に短く表示する。
